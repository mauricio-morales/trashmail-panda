## 2. Files to Modify (M)

### 2a. `src/TrashMailPanda/TrashMailPanda/TrashMailPanda.csproj`

**Remove** the following MSBuild constructs:

| Element | Action |
|---------|--------|
| `EnableAvaloniaUI` default property declaration | Delete entire conditional block |
| `PropertyGroup Condition="'$(EnableAvaloniaUI)' == 'true'"` (WinExe, ApplicationManifest, etc.) | Delete entirely |
| `PropertyGroup Condition="'$(EnableAvaloniaUI)' != 'true'"` (OutputType=Exe guard) | Simplify to unconditional `<OutputType>Exe</OutputType>` |
| `ItemGroup Condition="'$(EnableAvaloniaUI)' != 'true'"` (Compile Remove block) | Delete entirely (files no longer exist) |
| `ItemGroup Condition="'$(EnableAvaloniaUI)' == 'true'"` (AvaloniaResource) | Delete entirely |
| `ItemGroup Condition="'$(EnableAvaloniaUI)' == 'true'"` (Avalonia packages + CommunityToolkit.Mvvm) | Delete entirely |
